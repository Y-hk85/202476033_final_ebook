출처 표시와 적용

올바른 출처 작성, 함께 연습해요!

🌱 학습 목표

  • 다양한 저작물의 출처를 정확하게 표시할 수 있다.
  • 조건에 따른 출처 문장을 작성할 수 있다.
  • 출처 작성 후 평가 체크리스트로 자기 성찰을 할 수 있다.

📘 출처는 이렇게 써야 해요!

출처 작성에는 다음 요소들이 포함되어야 해요:
✔ 저작자 이름
✔ 작품 제목
✔ 라이선스 조건
✔ 원본 링크
예: 홍길동, 『숲속 야경』, CC BY-NC 4.0, https://original-link.com

예시 이미지는 여러분이 만든 창작물(혹은 직접 촬영한 사진) 이라고 가정해 봅시다. 이 작품을 공유하고 싶다면, 어떤 정보를 포함해 출처를 써야 할까요?

직접 출처 문장을 입력해 보며, 올바른 출처 표시 방법을 연습해 보세요.

  • 저작자 이름 (예: 김서연)
  • 작품 제목 (예: 『담요 속 고양이』)
  • 라이선스 조건 (예: CC BY-NC 4.0)
  • 원본 링크 또는 출처 URL (예: https://myblog.com/catphoto)

Tip: 모든 요소가 빠짐없이 들어가야 바른 출처랍니다!

📚 용어사전

저작권
사람이 만든 창작물(글, 그림, 음악 등)에 대해 창작자가 가지는 권리로, 다른 사람이 허락 없이 사용할 수 없게 보호해 주는 제도입니다.
라이선스
저작물을 사용할 때 지켜야 하는 약속이나 조건을 말합니다. 예를 들어, 'CC BY-NC'는 저작자를 밝히고, 비영리로만 사용할 수 있다는 뜻입니다.
출처
정보나 자료, 그림 등이 어디에서 왔는지 밝히는 것을 말합니다. 출처를 정확히 써야 저작권을 지킬 수 있습니다.
원본 링크
저작물의 실제 위치(웹사이트 주소 등)를 알려주는 인터넷 주소입니다.
